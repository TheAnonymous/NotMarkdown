# NotMarkdown licensing policy

NotMarkdown uses a permissive two-part licensing policy:

- Implementation code is licensed under the [MIT License](LICENSE).
- The format specification, schemas, grammar, conformance material, project
  documentation, and examples created by the NotMarkdown contributors are
  dedicated under [CC0 1.0 Universal](LICENSES/CC0-1.0.txt).

This split keeps code easy to embed in commercial and open-source products,
while allowing independent implementations to copy normative definitions,
test vectors, and examples without license friction. Files that contain an
explicit license header retain that header. Third-party material retains its
own license; see [THIRD_PARTY_NOTICES.md](THIRD_PARTY_NOTICES.md).

Documents authored by users are not covered by the project license merely
because they use the NotMarkdown format or its editors. Their authors retain
whatever rights apply to those documents.

Before a public release, contributors must confirm that they have the right to
offer every committed file under the applicable policy. This policy is a
project decision, not legal advice.
